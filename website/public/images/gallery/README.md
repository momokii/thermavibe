# Gallery Images

Place your demo images and GIFs here. They'll be served at `/images/gallery/`.

Recommended files:
- `demo-vibe-check.gif` — Screen recording of a vibe check session
- `demo-photobooth.gif` — Screen recording of a photobooth session
- `demo-admin.png` — Screenshot of the admin dashboard
- `demo-receipt.jpg` — Photo of a printed thermal receipt
- `demo-strip.jpg` — Photo of a printed photobooth strip

The gallery page references these via `src/data/gallery.ts`. Update the image
paths in that file to match your actual filenames.
